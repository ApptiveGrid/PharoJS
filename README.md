## Install

To install PharoJS, perform the code appropriate to your Pharo version in a playground:

### Pharo 9

<pre>
Metacello new
  baseline: 'PharoJS';
  repository: 'github://PharoJS/PharoJS:Pharo9Compatibility';
  load
</pre>

### Pharo 7 and Pharo 8

<pre>
Metacello new
  baseline: 'PharoJS';
  repository: 'github://PharoJS/PharoJS';
  load
</pre>
